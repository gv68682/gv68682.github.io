REAL TIME BUS TRACKER

From this application, we can track each bus status for a specific route. This project showcases how to fetch bus data using REST apis, filter required information, and place them on a map based on bus coordinates.

NOTE: Due to an API usage limitation, the app is automatically reset to allow only 4 updates every 15 seconds.
