REAL TIME BUS TRACKER

From this application, we can track each bus status for a specific route. We can see how to interpret json data to get latest updates for the buses. From this project, we can showcase how to fetch data using REST apis, filter required information, and place them on a map based on their coordinates.

NOTE: Due to API usage limitations app is made (resets on it's own) to allow only 3 updates.
